## Installation

[Available in Hex](https://hex.pm/docs/publish), so the package can be installed
by adding `ex_bin` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ex_bin, "~> 0.3.0"}
  ]
end
```
